Kindle highlights to markdown

Export your Kindle highlights.

One bookmarklet click on Kindle Notebook downloads every highlight and note you have into tidy markdown files, one book per file, ready for local tools and plain text workflows.

Step 1 — Drag this into your bookmarks bar

Export Kindle Highlights

Keep the button in your browser bookmarks bar so you can run the export from Kindle Notebook.

  1. Open Kindle Notebook Visit read.amazon.com/notebook while logged in to Amazon.
  2. Click the bookmarklet Use Export Kindle Highlights from your bookmarks bar to start the scrape and export flow.
  3. Download your zip A zip file downloads automatically with one markdown file per book.

Sample markdown output
